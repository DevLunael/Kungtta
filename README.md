# KKutuReact
> 끄투 React 리메이크 프로젝트

## Modules
KKutuReact는 구성요소를 3가지로 구별합니다.
- Client
- GameServer 
- API Server(OAuth, Common HTTP API)

## Tech Stack

### Client
- React.js (with create-react-app)
- TypeScript
- styled-components
- use-ws

### GameServer
- TypeScript
- Express
- Sequelize.js
- ws
- TSLint

### API Server
- TypeScript
- Express
- Passport.js
- Sequelize.js
- TSLint
